# FlatScouts Landingpage

Moderne, minimalistische Landingpage für das KI-Tool im Immobilien-Dokumentenmanagement.

## Technologie

- Next.js 14
- TypeScript
- CSS Modules

## Installation

```bash
npm install
```

## Entwicklung

```bash
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Build

```bash
npm run build
npm start
```

## Design

- **Fonts**: Ubuntu (Bold für Headlines), Inter Light (Normaler Text)
- **Farben**: 
  - Primär: #123c36
  - Secondary: #88a38c
  - Accent: #e3c9a5
  - Background: #F6EDE3
- **Stil**: Minimalistisch, viel Whitespace, 16px Border-Radius, mobil-first
